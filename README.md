# Human x Agent Collaboration Framework

A one-page visual framework by InnerG Intel on how humans and AI agents should collaborate.

## What this is
This project turns the Human x Agent Collaboration Framework into a clean visual doc that explains:

- the human role vs the agent role
- the 5-part collaboration formula
- better prompting habits
- better correction habits
- operating modes for human-agent systems
- common mistakes that slow collaboration down
- the InnerG standard for refined human-agent workflow
